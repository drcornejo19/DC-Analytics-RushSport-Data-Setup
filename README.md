# DC-Analytics-RushSport-Data-Setup

This project simulates a real-world commercial data setup process for an early-stage sports retail business.

## Objective
Build a realistic end-to-end demo showing how DC Analytics helps businesses structure their data from day one.

## Business Case
RushSport is a startup that sells sportswear and accessories through channels like WhatsApp, Instagram, MercadoLibre, and referrals.  
The business has growing sales, but its commercial data is unstructured, inconsistent, and difficult to analyze.

## Project Scope
- Generate a realistic raw dataset with Python
- Simulate messy operational data
- Audit and clean records
- Structure business entities
- Build a relational model
- Create KPIs and a dashboard

## Tech Stack
- Python
- Pandas
- NumPy
- Jupyter Notebook
- Power BI

## Repository Structure
- `data/raw/`: raw simulated data
- `data/processed/`: cleaned and structured data
- `notebooks/`: analysis and transformation notebooks
- `src/`: Python scripts
- `dashboard/`: BI assets
- `docs/`: documentation
- `images/`: visual assets
